# 컴퓨팅사고와 SW코딩

이 repository는 **컴퓨팅사고와 SW코딩** 수업에서 진행한 실습, 과제, 강의자료, Git/GitHub 연습 내용을 정리한 저장소입니다.

## Repository Purpose

본 저장소는 수업에서 수행한 Python 실습과 Jupyter Notebook 기반 과제 제출을 위해 구성되었습니다.

- Python 기초 문법 실습
- Jupyter Notebook 기반 LAB 및 연습문제 풀이
- Coding every day 학습 기록
- Git 및 GitHub 연동 연습
- 강의자료 및 시험 관련 자료 정리
