## Lab 3. return 반환값과 튜플

### 학습 목표
- `return` 문으로 함수의 결과값을 반환할 수 있다.
- 다중 반환값을 튜플로 받을 수 있다.
- 반환값을 변수에 저장하여 추가 연산에 활용할 수 있다.

### 배경

```python
# return이 없는 함수 — 결과를 재사용할 수 없음
def add_print(a, b):
    print(a + b)   # 출력만 하고 끝

# return이 있는 함수 — 결과를 변수에 저장하거나 연산에 사용 가능
def add_return(a, b):
    return a + b   # 값을 반환

result = add_return(3, 5)
print(result * 2)   # 16
```

```
1. 종료 조건 (Base Case)   → 재귀 호출을 멈추는 조건
2. 재귀 호출 (Recursive Call) → 자기 자신을 더 작은 문제로 호출
```

```python
def countdown(n):
    if n == 0:            # ① 종료 조건
        print("발사!")
        return
    print(n)
    countdown(n - 1)      # ② 재귀 호출 (문제가 작아짐)

countdown(5)
```
